# FASHION RECOMMENDER SYSTEM BY PROMPS

## Setup
```
$ conda create -n fclip python=3.11
$ conda activate fclip
$ pip install -r requirements_v2.txt
```

## Try your Prompt
```
$ streamlit run outfit_recommend_app_v2.py
```
